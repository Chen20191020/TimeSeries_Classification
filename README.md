# Structural Classification of Locally Stationary Time Series

This repository contains demonstration code for the TimeSeries_Classification project. It includes complete implementations of both simulation studies and real-data experiments used 


Our proposed method is described in the paper:

C. Qian, X. Ding, and L. Li. Structural Classification of Locally Stationary Time Series Based on Second-order Characteristics, arXiv: 2507.04237. https://arxiv.org/abs/2507.04237
